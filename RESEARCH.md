## Top inspiration set for the new build

1. **Wings Mobile Detailing** — conversion: immediate booking, “we come to you,” service confidence.
2. **Lakeside Mobile Detail** — local trust: service area, process steps, clean proof points.
3. **E1 Auto Spa / red-black premium category style** — visual identity: performance garage, redline lighting, glossy automotive energy.

## Design thesis

Build Easy E’s as a **mobile-first redline garage card**: the site should feel like Easton’s black/red business card became a high-energy smartphone landing page. It needs a memorable mechanic/detailing visual idea, not generic cards on a gradient.

Key difference from previous attempt: this version should be structurally different — a booking-first mobile landing page with a “detail ticket,” card-image proof, redline route, wash bay sections, process, pricing, and an action bar. It should not be a minor iteration of the old hero/pricing layout.
